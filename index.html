TypeScript

What is TypeScript?

  • TypeScript is a language developed by Microsoft.
  • TypeScript adds additional syntax to JavaScript.
  • TypeScript code converts to JavaScript, which runs anywhere JavaScript runs.
  • TypeScript understands JavaScript.

Advantages of TypeScript:

  • Enhances developer productivity: static typing helps catch errors during development.
  • Improves code support: IDEs offer more accurate suggestions and autocompletions thanks to types.
  • Provides scalability: TypeScript makes managing large projects easier due to typing.

Key Features of TypeScript:

  • Static Typing
  • Type Annotations
  • Interfaces
  • Enums

Static Typing


            function addNumbers(a: number, b: number): number {
                return a + b;
            }

            const result = addNumbers('10', 20);

Error: argument of type 'string' is not assignable to parameter of type 'number'.

Interfaces


            interface Person {
                name: string;
                age: number;
                greet(): string;
            }
  
            class Employee implements Person {
              constructor(public name: string, public age: number) {}
          
              greet(): string {
                  return `Hi, my name is ${this.name}`;
              }
          }

Enums


            enum DaysOfWeek {
                Monday,
                Tuesday,
                Wednesday,
                Thursday,
                Friday,
                Saturday,
                Sunday,
            }

            const today: DaysOfWeek = DaysOfWeek.Wednesday;
            const str = (`Today is ${DaysOfWeek[today]}.`);
